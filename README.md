# AR View

AR View is an augmented reality mobile application designed to bridge digital interaction with physical environments by allowing users to visualize and interact with 3D objects in real-world space.

The application focuses on creating immersive user experiences through real-time camera tracking, spatial object placement, and interactive visualization. This project explores how augmented reality can improve human-device interaction and create more intuitive digital experiences beyond traditional mobile interfaces.

## Features

* Real-time camera-based environment detection
* Interactive 3D object placement
* Dynamic object visualization in physical space
* Mobile optimized augmented reality interaction

## Technologies Used

* Unity
* C#
* AR SDK / ARCore
* Mobile Application Development

## Goal

To explore immersive computing and understand how augmented reality systems can create interactive experiences that merge digital and physical environments.
